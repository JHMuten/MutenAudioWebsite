Installation instructions and additional tips

Installation

  • Make sure your plugin is compatible with your computer, details should be found in the README within the zip download
  • Download and copy the VST3 file into the standard VST3 file location.
  • For windows the standard location is C:\Program Files\Common Files\VST3
  • You may have defined your own VST plugin location, double check in your DAW settings if you think this is the case, and manually place your new Muten Audio plugin in the new location
  • When you open your DAW it should automatically scan the plugin location, and the new plugin will be listed in the DAW's plugin section

Troubleshooting

If you encounter an issue when installing or using your plugin, don't hesitate to send an email to james@mutenaudio.co.uk. Please include as much information as you can about the computer system you are using, and any details about the problem.
